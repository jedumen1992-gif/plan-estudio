# Plan de estudio - GitHub Pages

Este paquete está listo para subir a GitHub Pages.

## Archivos
- index.html
- semana2_dia1_v3.html
- semana2_dia2_v3.html

## Pasos rápidos
1. Crear una cuenta en GitHub si no tenés.
2. Crear un repositorio nuevo, por ejemplo: plan-estudio
3. Subir estos archivos al repositorio (en la raíz).
4. Ir a Settings > Pages
5. En "Build and deployment":
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
6. Guardar.
7. GitHub te va a dar un link parecido a:
   https://TU-USUARIO.github.io/plan-estudio/

## Importante
- Dejá los archivos con esos mismos nombres.
- Cada computadora / navegador guarda su progreso por separado.
